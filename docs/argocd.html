GitOps is an operational framework that takes DevOps best practices used for application development‚Äîsuch as version control, collaboration, compliance, and CI/CD‚Äîand applies them to infrastructure automation.

App-of-Apps Pattern

Managing multiple applications as a single ArgoCD application. This allows for bootstrapping entire clusters or environment-specific toolsets with a single click. 

apiVersion: argoproj.io/v1alpha1
kind: Application
metadata:
  name: root-app
spec:
  source:
    path: apps/production
    repoURL: $GIT_REPO
