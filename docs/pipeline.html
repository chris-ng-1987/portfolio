Basic Node.js App

.gitlab-ci.yml
# Include the ready-to-use template
include:
  - project: 'hub/gitlab/ci-template'
    file: '/ci-templates/nodejs-common.yml'

# Override variables for your app

  APP_DIR: ./my-app
  CONTAINER_IMAGE: $CI_REGISTRY_IMAGE:$CI_COMMIT_SHA

Extending Job Templates

Add custom jobs using templates

.gitlab-ci.yml
include:
  - project: 'hub/gitlab/ci-template'
    file: '/job-templates/kaniko-build.yml'
  - project: 'hub/gitlab/ci-template'
    file: '/job-templates/argocd-deploy.yml'

# Extend and customize
build-app:
  extends: .kaniko-build
  variables:
    DOCKERFILE: Dockerfile.prod

deploy-staging:
  extends: .argocd-dev-deploy
  variables:
    ARGOCD_APP_NAME: my-app-staging

ArgoCD GitOps Deploy

Sync and wait for deployment

argocd-deploy.yml
.argocd-deploy:
  stage: deploy
  script:
    # Create/update ArgoCD application
    - argocd app create --upsert -f $ARGOCD_APP_FILE
    
    # Sync and wait for rollout
    - argocd app sync $APP_NAME --timeout 600
    - argocd app wait $APP_NAME --grpc-web
    
    # Print deployment logs
    - argocd app logs $APP_NAME --kind=Deployment
